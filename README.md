# Ujian SIT

**Ujian SIT** merupakan sebuah proyek aplikasi ujian berbasis web yang dikembangkan untuk mendukung pelaksanaan ujian secara digital. Proyek ini berada di bawah organisasi [Tim-Fakultas](https://github.com/Tim-Fakultas) dan memiliki tujuan utama untuk memudahkan pengelolaan, pelaksanaan, serta evaluasi ujian bagi institusi pendidikan maupun organisasi yang membutuhkan solusi ujian online yang fleksibel dan aman.

## Fitur Utama
- Pembuatan dan pengelolaan bank soal
- Manajemen peserta ujian
- Penjadwalan dan pelaksanaan ujian secara online
- Monitoring hasil ujian secara real-time
- Otentikasi dan keamanan data ujian

## Cara Instalasi & Pengembangan

1. **Clone repository**
   ```bash
   git clone https://github.com/Tim-Fakultas/ujian-sit.git
   ```
2. **Instal dependencies**
   - Untuk backend & frontend, sesuaikan dengan dokumentasi pada masing-masing folder (`/backend`, `/frontend`, dsb).
3. **Setting environment**
   - Atur file `.env` sesuai kebutuhan pengembangan lokal.
4. **Jalankan aplikasi**
   - Ikuti instruksi pada dokumentasi masing-masing package untuk menjalankan aplikasi secara lokal.
